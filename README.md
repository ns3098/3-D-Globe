
# 3-D Globe

### Installing (Front-End)

1. Install dependencies
```sh
npm install
```

2. Fire up the server and watch files
```sh
npm run dev
```

### Installing (Back-End)

1. Go to backend folder
```sh
cd backend
```

2. Install dependencies
```sh
npm install
```
3. Start the server
```sh
npm start
```

### Compiles and minifies for production
```sh
npm run build
```
